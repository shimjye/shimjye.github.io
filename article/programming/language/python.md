## python-env
- 설치 $ pip3 install virtualenv
- 환경생성 $ virtualenv .env
- 기본환경 $ virtualenv -p python3 [env_name]
- $ source .env/bin/activate
- $ deactivate

## pip
- $ pip3 install requests beautifulsoup4
- $ pip3 freeze > requirements.txt
- $ pip3 install -r requirements.txt
